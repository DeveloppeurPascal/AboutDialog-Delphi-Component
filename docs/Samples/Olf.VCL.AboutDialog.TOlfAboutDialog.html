Properties

Copyright

Copyright du projet, affiché sous le numéro de version

Description

Texte de description à afficher dans la boite de dialogue S'il est vide, son pavé n'est pas affiché. S'il est indiqué et qu'une licence est aussi présente, ils se partagent l'espace disponible.

Image

Image utilisée comme logo dans la boite de dialogue (pris en compte si pas d'ImageList)

ImageIndex

Index de l'image provenant de la liste d'images correspondant au logo à utiliser dans la boite de dialogue

ImageName

Nom de l'image provenant de la liste d'images correspondant au logo à utiliser dans la boite de dialogue (le nom est prioritaire sur l'index s'il est renseigné et géré par la liste d'images)

Images

Liste d'images contenant le logo à utiliser dans la boite de dialogue

Langue

Langue des textes affichés par la boite de dialogue

Licence

Texte de la licence à afficher dans la boite de dialogue S'il est vide, son pavé n'est pas affiché. S'il est indiqué et qu'une description est aussi présente, ils se partagent l'espace disponible.

MainFormCaptionPrefix

Used in the main form caption string between "[DEBUG]" (if any) and the program title

MainFormCaptionSuffix

Used in the main form caption after the title and program version

Picture

Bitmap utilisé par l'image servant de logo dans la boite de dialogue (pris en compte si pas d'ImageList ni d'Image)

ReplaceMainFormCaption

Allow replacing the main form caption by the Aboutbox component. If "True", changes in MainFormCaptionPrefix, MainFormCaptionSuffix, Titre, VersionNumero erase current main form caption of the project.

Titre

Titre affiché dans la barre de titre de la fenêtre "à propos" et en premier sur la boite de dialogue

URL

Adresse du site pour lequel on affiche un lien sous le numéro de version/date

VersionDate

Affiché sous le titre sur la forme "Version du JJ/MM/AAAA" ou "Version XXX du JJ/MM/AAAA"

VersionNumero

Affiché sous le titre sur la forme "version XXX" ou "Version XXX du JJ/MM/AAAA"
